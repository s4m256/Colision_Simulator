# Colision Simulator

Interactive particle collision simulator developed in **p5.js**.  
Visualize and study collisions in 2D while adjusting parameters like mass, velocity, and elasticity.

---

## Features

- Collision detection and response.
- Real-time visualization of particle trajectories.
- Easy configuration of particle mass and speed.
- Fit in any screen
---

## Technologies

- **JavaScript**
- **p5.js** for graphics and basic physics.
